# 同步机制

## 当前范围（最小化）

### 1. 自我优化进度
- 记录各助理的迭代进度
- 同步技能更新状态
- 追踪优化效果

### 2. 定时任务信息
- 定时抓取的信息汇总
- 任务执行状态
- 异常和告警

## 同步文件

```
sync/
├── status/                # 状态文件目录
│   ├── max-status.md      # Max 状态
│   ├── h-status.md        # H 状态
│   └── eva-status.md      # Eva 状态
├── tasks/                 # 定时任务目录
│   └── task-log.md        # 任务执行日志
└── scripts/               # 同步脚本
    ├── push.sh            # 推送状态
    └── pull.sh            # 拉取状态
```

---

## 状态文件格式

### status/{name}-status.md

```markdown
# {Name} 状态

## 最后更新时间
2026-04-27 12:00:00

## 当前状态
- 运行中 / 空闲 / 异常

## 迭代进度
| 项目 | 状态 | 完成度 |
|------|------|--------|
|      |      |        |

## 最近更新
- 2026-04-27: ...
```
